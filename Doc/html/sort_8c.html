#define I   55
 Constant that sets the highest number in the array we want to sort and also the number of lines there will be on the plot. Must be smaller or equal to the number of lines of the terminal.

 Constant that sets the number of values to sort and also the number of column there will be on the plot. Must be smaller or equal to the number of column of the terminal.
